A smart contract is a self-executing program stored on a blockchain. Here's what you need to know:

**What it is:** A smart contract is code deployed to a blockchain (like Ethereum) that automatically executes when conditions are met. It's like a digital agreement that runs itself without needing a middleman.

**Key features:**
- **Immutable:** Once deployed, the code can't be changed (unless the contract explicitly has upgrade logic).
- **Transparent:** Anyone can read the code and verify what it does.
- **Autonomous:** It executes automatically when triggered, without human intervention.
- **Deterministic:** Given the same input, it always produces the same output.

**Real-world examples:**
- A token swap: "If User sends USDC, automatically send them ETH at the current rate."
- A lending protocol: "If borrower's collateral drops below 150%, automatically liquidate their position."
- An NFT auction: "When time expires, send the NFT to highest bidder and payment to seller."

**How it differs from traditional code:**
- Runs on decentralized infrastructure (multiple nodes) rather than a single server
- State changes are permanent and visible to everyone
- Executes with cryptographic certainty — no one can fake the result
