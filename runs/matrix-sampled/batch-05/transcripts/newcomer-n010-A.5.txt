[FINAL_REPLY_TO_USER]
There are several ways to verify a wallet is legitimate:

1. **Official websites only** — Download from the official site, never from random links. For example, MetaMask's official site is metamask.io (not metamask-io.com or similar variants).

2. **Check the domain carefully** — Typosquatters use lookalike domains like:
   - metmask.io (swapped letters)
   - metamask.io.io (double extension)
   - meta-mask.com (dash variations)
   Always verify the exact domain matches before entering any credentials.

3. **Verify with community sources** — Check official Twitter/X accounts, official GitHub repos, and reputable review sites like CoinGecko or Trustwallet's official documentation.

4. **Look for code transparency** — Legitimate wallets like Ledger Live, Exodus, and MetaMask publish their source code on GitHub so security researchers can audit them.

5. **Hardware wallets add security** — If you're holding significant amounts, a hardware wallet like Ledger or Trezor is more secure than a software wallet, since private keys never leave the device.

6. **Never trust shortened URLs or link redirects** — Always navigate to the full domain directly from a search engine or saved bookmark.

7. **Beware of unsolicited recovery prompts** — If a website suddenly asks you to recover your wallet when you didn't request it, it's almost certainly a phishing site.

The golden rule: If you're unsure, the safest option is to use a well-established wallet with a long track record of security (MetaMask, Ledger, Trezor, Exodus) and always verify the official domain.
